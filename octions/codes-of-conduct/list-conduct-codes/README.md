<a name="quick start" ></a>
## Quick start

```yaml
- uses: maxkomarychev/octions/octions/codes-of-conduct/list-conduct-codes@master
  id: my_step_id
  with:
    token: <token value>
- name: Print outputs
  run: |
    echo ${{ steps.my_step_id.outputs.id }}
    echo ${{ steps.my_step_id.outputs.number }}
```
